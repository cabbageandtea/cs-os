# Security Policy

## Reporting a Vulnerability

Report potential security issues to hello@doggybagg.cc.

Please include:
- A clear description of the issue
- Reproduction steps and impact
- Any proof-of-concept details needed to validate

## Scope

This policy applies to:
- doggybagg.cc and subdomains managed by this project
- The source code in this repository

## Responsible Disclosure Expectations

- Do not publicly disclose vulnerabilities before coordinated remediation
- Do not access, modify, or exfiltrate data that is not your own
- Avoid actions that impact availability for real users

We will review reports, acknowledge valid findings, and coordinate remediation updates.
