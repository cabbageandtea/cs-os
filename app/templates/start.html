Why students claim a .me domain

Recruiters skim hundreds of resumes. yourname.me is one link they can type from memory — same URL on your PDF, LinkedIn featured links, and applications. A GitHub Pages path works, but it looks like coursework hosting until you graduate.

  • Memorable — “about me” built into the TLD
  • Aligned — we put the same URL on resume + LinkedIn during delivery
  • Often free — GitHub Education → Student Pack → nc.me (when eligible)

Not a student or no pack yet? Every package deploys to github.io. Upgrade to .me anytime.
