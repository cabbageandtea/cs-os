# Data-Tech portfolio starter

Engineering-forward layout: stack sidebar, outcome-led project cards.

## When to use

- SWE, data, analytics, DevOps targets
- Foundation or Launch packages with technical proof-of-work

## Placeholders

See `minimal/README.md` for shared tokens plus:

| Token | Notes |
|-------|-------|
| `PROJECT_*_PROBLEM` | What was broken / missing |
| `PROJECT_*_OUTCOME` | Measurable or demonstrable result |
| `PROJECT_*_DEMO` / `*_REPO` | Live URL + GitHub |
| `STACK_*` | Primary technologies |

## Deploy

Same as minimal — GitHub Pages or Render static site.
