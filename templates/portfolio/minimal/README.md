# Minimal portfolio starter

Copy this folder into the client's repo. Replace `{{PLACEHOLDER}}` tokens from intake.

## Slots

| Token | Source |
|-------|--------|
| `FULL_NAME` | Intake name |
| `TARGET_ROLE` | Intake target role |
| `ONE_LINE_PITCH` | Career goals / summary |
| `ABOUT_PARAGRAPH` | Education + work compressed |
| `PROJECT_*` | Intake projects section |
| `SKILL_*` | Intake skills |
| `EMAIL` | Client email |
| `GITHUB_URL` / `LINKEDIN_URL` | Intake links |

## Deploy

1. Push to `username.github.io` repo or enable GitHub Pages on project repo
2. Set custom domain in repo settings if Accelerator package
