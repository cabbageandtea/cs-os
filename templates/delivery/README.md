# Delivery templates

Operator-fillable docs for each package deliverable. Placeholders like `{{CLIENT_NAME}}` are filled on the client detail page.

| File | Deliverable | Packages |
|------|-------------|----------|
| `github-profile-guidance.md` | GitHub profile guidance | All |
| `deployment-handoff.md` | Deployment URL | All |
| `linkedin-optimization-notes.md` | LinkedIn optimization notes | Launch, Accelerator |
| `resume-build-guide.md` | Resume (PDF) | Launch, Accelerator |
| `career-narrative-summary.md` | Career narrative summary | Accelerator |
| `strategy-session-agenda.md` | Strategy session (30 min) | Accelerator |
| `custom-domain-setup-guide.md` | Custom domain setup guide | Accelerator |

**Portfolio website:** build from `templates/portfolio/` (see client detail Build kit).  
**Resume HTML starter:** `templates/resume/starter.html`

Registry: `app/delivery_kits.py`
